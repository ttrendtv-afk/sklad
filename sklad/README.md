# 📱 Склад Телефонов — Инструкция по установке

## Что это?
Полноценная веб-программа для управления складом телефонов.
- Вход с паролем для каждого сотрудника
- Добавление телефонов на склад
- Продажа с расчётом прибыли
- История всех операций
- Отчёты и статистика
- Доступ с любого устройства через интернет

---

## 🚀 Установка на Railway (бесплатно, без терминала)

### Шаг 1 — GitHub
1. Зайди на https://github.com и зарегистрируйся (бесплатно)
2. Нажми кнопку **"New repository"** (зелёная кнопка)
3. Назови репозиторий: `sklad-telefonov`
4. Нажми **"Create repository"**

### Шаг 2 — Загрузи файлы
1. На странице репозитория нажми **"uploading an existing file"**
2. Перетащи ВСЕ файлы из папки `sklad` в браузер
3. Нажми **"Commit changes"**

### Шаг 3 — Railway
1. Зайди на https://railway.app
2. Нажми **"Login with GitHub"**
3. Нажми **"New Project"** → **"Deploy from GitHub repo"**
4. Выбери твой репозиторий `sklad-telefonov`
5. Railway сам установит всё и запустит!

### Шаг 4 — Получи ссылку
1. В Railway нажми на проект → вкладка **"Settings"**
2. Найди **"Generate Domain"** — нажми
3. Получишь ссылку типа `https://sklad-telefonov.up.railway.app`

## 💻 Локальная установка (офлайн, на компьютере)

### Нужно установить Python:
1. Скачай Python с https://python.org (кнопка Download)
2. При установке отметь галочку **"Add Python to PATH"**

### Запуск:
1. Открой папку с файлами
2. Дважды кликни на файл `start.bat` (Windows)
3. Открой браузер и перейди на http://localhost:5000

## ❓ Частые вопросы

**Данные сохраняются?**
Да, в базе данных SQLite (файл sklad.db). Не удаляй этот файл!

**Могут работать несколько человек одновременно?**
Да, до 5 человек без проблем.

**Можно использовать и онлайн и офлайн?**
Онлайн — через Railway (доступ из любой точки мира).
Офлайн — через локальный запуск (только в вашей сети).
